MonteCarloMeasurementUncertainty (MCMU)

This package converts a (correlated) data stream into a measurement with a value and error with the help of the OnlineLogBinning.jl and Measurements.jl packages.

Add this package from Julia's General Registry in any the usual way. For example, in the Julia REPL, simply hit ] to open the pkg manager hosted by Pkg:

pkg> add MonteCarloMeasurementUncertainty

Then hit Backspace to exit pkg mode. And voila! You now have access to MCMU!
